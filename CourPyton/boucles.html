Boucle for en Python

Une introduction détaillée à la boucle for en Python pour itérer sur des collections de données.

1. La Boucle for

La boucle for est utilisée pour itérer sur des éléments d'une collection, comme des listes, des tuples, des ensembles ou des dictionnaires. Elle permet de traiter chaque élément de la collection de manière répétée.

Exemple de boucle for avec une liste

personnes = ["Alice", "Bob", "Charlie"]
for personne in personnes:
    print(personne)

Sortie :

Alice
Bob
Charlie

Utilisation de range() avec for

La fonction range() génère une séquence de nombres, souvent utilisée pour répéter une action un certain nombre de fois.

for i in range(3):
    print(i)

Sortie :

0
1
2

Itérer sur un dictionnaire

Il est possible d'itérer sur un dictionnaire en accédant aux clés ou aux valeurs avec la boucle for.

personne = {"nom": "Alice", "age": 25, "ville": "Paris"}
for cle, valeur in personne.items():
    print(cle, ":", valeur)

Sortie :

nom : Alice
age : 25
ville : Paris

La boucle for avec des conditions

Vous pouvez également ajouter des conditions dans la boucle for pour ne traiter certains éléments que s'ils satisfont une condition.

nombres = [1, 2, 3, 4, 5]
                for nombre in nombres:
                if nombre % 2 == 0:
                 print(nombre)  # Affiche uniquement les nombres pairs

Sortie :

2
            4

Utilisation de range()

La fonction range() est couramment utilisée dans les boucles for pour générer une séquence d'entiers. Contrairement aux tuples, qui stockent des éléments fixes, range() génère des valeurs à la volée, ce qui le rend plus efficace en mémoire lorsqu'on travaille avec de longues séquences de nombres.

Exemple de boucle avec range() :

for i in range(5):
                print(i)

Ce code affichera :

0
            1
            2
            3
            4

Détails sur range() :

  • Non mutable : Un objet range est immuable, il ne peut pas être modifié après sa création.
  • Générateur d'entiers : range() génère des entiers dans une séquence spécifiée, mais ne stocke pas ces entiers en mémoire.
  • Utilisation en boucle : range() est souvent utilisé pour spécifier combien de fois une boucle for doit s'exécuter.

Exemple de range() avec des paramètres :

for i in range(2, 10, 2):
                print(i)

Cela affiche :

2
            4
            6
            8

Ici, la séquence commence à 2, se termine avant 10, et s'incrémente de 2 à chaque itération.

Différence avec les tuples :

Contrairement à un tuple (qui contient des éléments prédéfinis et stockés en mémoire), range() crée les valeurs à la volée pour économiser de la mémoire, ce qui est particulièrement utile dans les boucles.

Un tuple est une structure de données statique, tandis qu'un range() est principalement un outil pour itérer et travailler avec des séquences d'entiers dans les boucles.
