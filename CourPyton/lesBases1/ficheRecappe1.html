Documentation Python

Types de Variables

Les types de données les plus utilisés en Python :

a = "Bonjour" # chaine de caractères : str
b = 5 # nombre entier : int
c = 1.5 # nombre à virgules : float
d = True # True ou False : bool

Print et Input

Comment afficher et récupérer des données :

nom = input("Quel est votre nom? ")
print("Vous vous appelez " + nom) # concaténation de chaine
print(f"Vous vous appelez {nom}") # chaine formatée
print("Vous vous appelez %s" % nom) # chaine formatée (ancien format)

Conversions

Convertir entre les types de données :

age = 30
print("Votre age est: " + str(age)) # conversion de int vers str, et concaténation
age_str = "30"
age_int = int(age_str) # conversion de str vers int.

Utiliser un Bloc try/except

Utiliser un bloc `try/except` pour gérer les erreurs :

try:
  age_int = int(age_str)
except:
  print("ERREUR: Vous devez rentrer un nombre pour l'age")

Boucle While

Une boucle qui s'exécute tant que la condition est vraie :

nom = ""
while nom == "":
  nom = input("Quel est votre nom? ")

Boucle For

Une boucle qui s'exécute un nombre de fois défini :

for i in range(0, 4): # de 0 (inclus) à 4 (exclu) : 0, 1, 2, 3
  print(i)

Conditions

Utiliser des conditions pour exécuter du code selon les critères donnés :

if age == 17:
  print("Vous êtes presque majeur")
elif 12 <= age < 18:
  print("Vous êtes adolescent")
elif age == 1 or age == 2:
  print("Vous êtes un bébé")
elif age >= 18:
  print("Vous êtes majeur")
else:
  print("Vous êtes mineur")
