Les Collections en Python

Une introduction détaillée aux collections de données en Python : Listes, Tuples, Ensembles, Dictionnaires, et plus encore.

1. Listes

Les listes en Python sont des collections ordonnées et modifiables. Elles permettent de stocker des éléments de différents types et de les manipuler grâce à diverses méthodes intégrées.

Exemple de liste simple

personnes = ["Alice", "Bob", "Charlie"]

Modification d'une liste


            personnes[1] = "Eve"
            personnes.append("David")
            print(personnes)

Sortie :

['Alice', 'Eve', 'Charlie', 'David']

Méthodes courantes des listes

  • append(élément): Ajoute un élément à la fin de la liste. 
    
            fruits = ["pomme", "banane"]
            fruits.append("orange")
            print(fruits)
            # Sortie : ['pomme', 'banane', 'orange']
  • insert(index, élément): Insère un élément à une position spécifique. 
    
            fruits = ["pomme", "orange"]
            fruits.insert(1, "banane")
            print(fruits)
            # Sortie : ['pomme', 'banane', 'orange']
  • remove(élément): Supprime la première occurrence d'un élément. 
    
            fruits = ["pomme", "banane", "orange"]
            fruits.remove("banane")
            print(fruits)
            # Sortie : ['pomme', 'orange']
  • pop([index]): Supprime et retourne un élément à un indice spécifique. Si aucun indice n'est précisé, retire le dernier élément. 
    
            fruits = ["pomme", "banane", "orange"]
            dernier = fruits.pop()
            print(dernier)  # Sortie : orange
            print(fruits)
            # Sortie : ['pomme', 'banane']
  • clear(): Supprime tous les éléments de la liste. 
    
            fruits = ["pomme", "banane", "orange"]
            fruits.clear()
            print(fruits)
            # Sortie : []
  • index(élément[, début, fin]): Retourne l'indice de la première occurrence d'un élément. Peut lever une erreur si l'élément n'est pas trouvé. 
    
            fruits = ["pomme", "banane", "orange"]
            indice = fruits.index("banane")
            print(indice)
            # Sortie : 1
  • count(élément): Retourne le nombre d'occurrences d'un élément dans la liste. 
    
            fruits = ["pomme", "banane", "pomme", "orange"]
            nb_pommes = fruits.count("pomme")
            print(nb_pommes)
            # Sortie : 2
  • sort(key=None, reverse=False): Trie les éléments de la liste en place. Vous pouvez spécifier un paramètre key pour personnaliser le tri. 
    
            nombres = [5, 2, 9, 1]
            nombres.sort()
            print(nombres)
            # Sortie : [1, 2, 5, 9]
            
            nombres.sort(reverse=True)
            print(nombres)
            # Sortie : [9, 5, 2, 1]
  • reverse(): Inverse l'ordre des éléments de la liste. 
    
            nombres = [1, 2, 3, 4]
            nombres.reverse()
            print(nombres)
            # Sortie : [4, 3, 2, 1]
  • copy(): Crée une copie superficielle de la liste. 
    
            fruits = ["pomme", "banane", "orange"]
            copie = fruits.copy()
            print(copie)
            # Sortie : ['pomme', 'banane', 'orange']
  • extend(itérable): Ajoute les éléments d'un itérable (liste, tuple, etc.) à la liste actuelle. 
    
            fruits = ["pomme", "banane"]
            autres_fruits = ["orange", "kiwi"]
            fruits.extend(autres_fruits)
            print(fruits)
            # Sortie : ['pomme', 'banane', 'orange', 'kiwi']

Autres opérations sur les listes

Voici quelques opérations utiles :

  • len(liste): Retourne le nombre d'éléments dans la liste. 
    
            fruits = ["pomme", "banane", "orange"]
            taille = len(fruits)
            print(taille)
            # Sortie : 3
  • in et not in: Vérifient si un élément est présent ou absent dans une liste. 
    
            fruits = ["pomme", "banane", "orange"]
            print("banane" in fruits)  # Sortie : True
            print("kiwi" not in fruits)  # Sortie : True
  • + et *: Concatenation et répétition de listes. 
    
            fruits = ["pomme", "banane"]
            autres_fruits = ["orange", "kiwi"]
            print(fruits + autres_fruits)
            # Sortie : ['pomme', 'banane', 'orange', 'kiwi']
            
            print(fruits * 2)
            # Sortie : ['pomme', 'banane', 'pomme', 'banane']

Découper une chaîne de caractères avec split()

La méthode split() permet de découper une chaîne de caractères en une liste de sous-chaînes en utilisant un séparateur spécifié.

  • split(sep=None, maxsplit=-1) : Découpe la chaîne en une liste de sous-chaînes, où sep est le séparateur (par défaut l'espace) et maxsplit définit le nombre maximum de découpes. 
    
    phrase = "Bonjour tout le monde"
    mots = phrase.split()
    print(mots)
    # Sortie : ['Bonjour', 'tout', 'le', 'monde']
  • Si vous souhaitez découper la chaîne en fonction d'un caractère spécifique (par exemple, une virgule), vous pouvez le spécifier en argument : 
    
    fruits = "pomme,banane,orange"
    fruits_list = fruits.split(",")
    print(fruits_list)
    # Sortie : ['pomme', 'banane', 'orange']

1.1FONCTION-LISTES

Les fonctions en Python permettent de manipuler les listes ou d'autres structures telles que les tuples. Voici un exemple d'utilisation de fonctions qui retournent des tuples et comment les exploiter.

Exemple avec une fonction qui retourne un tuple


        def obtenir():
            return "melain", 37, 1.60
        
        # Déstructuration du tuple directement dans des variables
        nom, age, taille = obtenir()
        print(f"nom: {nom}, age: {age}, taille: {taille}")

Exemple avec deux fonctions


        def obtenir():
            return "melain", 37, 1.60
        
        def afficher_info(nom, age, taille):
            print(f"nom: {nom}, age: {age}, taille: {taille}")
        
        # Appel des fonctions
        nom, age, taille = obtenir()
        afficher_info(nom, age, taille)

Autre méthode


        def obtenir():
            return "melain", 37, 1.60
        
        def afficher_info(nom, age, taille):
            print(f"nom: {nom}, age: {age}, taille: {taille}")
        
        # Utilisation de l'opérateur de déstructuration (*)
        infos = obtenir()
        afficher_info(*infos)

Ces exemples montrent comment exploiter les fonctions pour retourner et manipuler des tuples en Python.

1.2 Les Slices

Les slices en Python permettent d'extraire une portion d'une séquence (comme une liste, une chaîne de caractères ou un tuple). Ils utilisent la syntaxe [début:fin:pas].

Syntaxe générale

  • début (optionnel) : L'indice où commencer la tranche (inclus).
  • fin (optionnel) : L'indice où terminer la tranche (exclus).
  • pas (optionnel) : L'incrément entre chaque élément.

Exemples


            # Exemple de slice simple
            liste = [0, 1, 2, 3, 4, 5]
            print(liste[1:4])  # Sortie : [1, 2, 3]
            
            # Slice avec un pas
            print(liste[::2])  # Sortie : [0, 2, 4]
            
            # Slice avec indices négatifs
            print(liste[-4:-1])  # Sortie : [2, 3, 4]
            
            # Slice inversé
            print(liste[::-1])  # Sortie : [5, 4, 3, 2, 1, 0]

Applications courantes

  • Extraire une sous-liste : 
    
            nombres = [10, 20, 30, 40, 50]
            sous_liste = nombres[1:4]
            print(sous_liste)
            # Sortie : [20, 30, 40]
  • Copier une liste : 
    
            nombres = [10, 20, 30]
            copie = nombres[:]
            print(copie)
            # Sortie : [10, 20, 30]
  • Inverser une liste : 
    
            nombres = [1, 2, 3]
            inverse = nombres[::-1]
            print(inverse)
            # Sortie : [3, 2, 1]

Travailler avec des chaînes


            texte = "Python"
            print(texte[1:4])  # Sortie : yth
            print(texte[::-1]) # Sortie : nohtyP

Travailler avec des tuples


            tuples = (10, 20, 30, 40)
            print(tuples[1:3])  # Sortie : (20, 30)
            print(tuples[::-1]) # Sortie : (40, 30, 20, 10)

Les slices sont puissants pour manipuler des séquences de manière concise et élégante.

2. Tuples

Les tuples sont des collections immuables, ce qui signifie que leurs éléments ne peuvent pas être modifiés après leur création. Ils sont définis par des parenthèses.

Exemple de tuple simple

personnes = ("Alice", "Bob", "Charlie")

Immuabilité des tuples

Les tuples ne peuvent pas être modifiés une fois créés. Tenter de modifier un élément d'un tuple entraînera une erreur.

personnes[1] = "Eve"  # Cela génère une erreur

Méthodes des tuples

  • count(): Compte combien de fois un élément apparaît dans un tuple.
  • index(): Renvoie l'indice du premier élément trouvé dans un tuple.

Parcourir un Tuple avec une Boucle for

Une méthode très pratique pour parcourir un tuple est d'utiliser une boucle for. Cela permet de récupérer chaque élément du tuple sans se soucier des indices. Voici un exemple :

personnes = ("Mélanie", "jean", "marti", "Alice")
        
        # Parcours du tuple avec une boucle for
        for i in personnes:
            print(i)
            print(len(i))  # Affiche la longueur de chaque élément du tuple
            print(i[0])    # Affiche le premier caractère de chaque élément du tuple

Sortie :

Mélanie
        7
        M
        jean
        4
        j
        marti
        5
        m
        Alice
        5
        A

Dans cet exemple, la boucle for parcourt chaque élément du tuple et, pour chaque élément :

  • print(i) affiche l'élément lui-même.
  • print(len(i)) affiche la longueur de l'élément, c'est-à-dire le nombre de caractères dans chaque nom.
  • print(i[0]) affiche le premier caractère de chaque élément du tuple.

4. Dictionnaires

Les dictionnaires

Utilité des Dictionnaires

Les dictionnaires sont utilisés pour stocker des données de manière structurée et efficace. Contrairement aux listes, qui utilisent des indices numériques pour accéder aux éléments, les dictionnaires utilisent des clés uniques pour accéder à leurs valeurs correspondantes. Cette approche permet un accès rapide et intuitif aux données, ce qui est particulièrement utile lorsque les relations entre les éléments ne sont pas strictement séquentielles. Voici quelques points clés de leur utilité :

Stockage Structuré

Les dictionnaires permettent de stocker des données de manière organisée, en utilisant des clés pour identifier chaque valeur. Cela simplifie l'accès et la gestion des données, surtout lorsque les données sont complexes ou imbriquées.

Recherche Rapide

Les dictionnaires offrent un accès très rapide aux données grâce à l'utilisation de clés uniques. Rechercher une valeur à l'aide de sa clé est beaucoup plus rapide que de parcourir une liste pour trouver un élément.

Flexibilité

Les dictionnaires peuvent contenir des données de types variés (entiers, chaînes de caractères, listes, autres dictionnaires, etc.). Cette flexibilité permet de modéliser des relations de données complexes de manière simple et intuitive.

Pourquoi utiliser des Dictionnaires ?

Simplicité et Lisibilité

Les dictionnaires rendent le code plus lisible et facile à comprendre en utilisant des clés significatives au lieu d'indices numériques. Cela aide à rendre le code auto-documenté et plus facile à maintenir.

Performance

En raison de leur implémentation basée sur des tables de hachage, les opérations de recherche, d'ajout et de suppression dans un dictionnaire sont très rapides. Cela rend les dictionnaires particulièrement efficaces pour les applications nécessitant des accès fréquents aux données.

Organisation des Données

Les dictionnaires sont idéaux pour organiser des données en paires clé-valeur. Cela est particulièrement utile dans des situations où chaque élément de donnée a une signification unique, comme un répertoire de contacts, où chaque contact a un nom unique (clé) et des informations associées (valeur).

Quand utiliser des Dictionnaires ?

Association Clé-Valeur

Utilisez des dictionnaires lorsque vous avez besoin d'associer des valeurs à des clés uniques. Par exemple, dans un répertoire téléphonique, vous pouvez utiliser les noms comme clés et les numéros de téléphone comme valeurs.

Données Non Séquentielles

Lorsque les données ne suivent pas un ordre séquentiel ou lorsque l'ordre n'a pas d'importance, les dictionnaires sont plus appropriés que les listes. Par exemple, pour stocker des paramètres de configuration, où chaque paramètre a un nom unique.

Manipulation de Données JSON

Les dictionnaires sont souvent utilisés pour manipuler des données au format JSON. Les structures JSON peuvent être directement converties en dictionnaires Python, ce qui facilite leur manipulation et leur traitement.

Comptage et Agrégation

Les dictionnaires sont utiles pour compter la fréquence des éléments dans une collection. Par exemple, pour compter le nombre d'occurrences de chaque mot dans un texte, vous pouvez utiliser les mots comme clés et leurs fréquences comme valeurs.

Modélisation de Graphes

Dans les algorithmes de graphes, les dictionnaires sont utilisés pour représenter les relations entre les nœuds. Les nœuds peuvent être représentés comme des clés et leurs voisins comme des valeurs.

Exemple de dictionnaire simple

personne = {"nom": "Alice", "age": 25, "ville": "Paris"}

Accès aux valeurs d'un dictionnaire

print(personne["nom"])  # Accès à la valeur de la clé "nom"

Sortie :

Alice

Méthodes des dictionnaires

  • get(): Accède à la valeur d'une clé sans lever d'erreur si la clé n'existe pas. 
    print(personne.get("profession", "Non spécifié"))  # Accède à la valeur de la clé "profession" ou renvoie "Non spécifié" si la clé n'existe pas.

    Sortie :

  • keys(): Renvoie toutes les clés d'un dictionnaire. 
    print(personne.keys())  # Renvoie toutes les clés du dictionnaire

    Sortie :

    dict_keys(['nom', 'age', 'ville'])
  • values(): Renvoie toutes les valeurs d'un dictionnaire. 
    print(personne.values())  # Renvoie toutes les valeurs du dictionnaire

    Sortie :

    dict_values(['Alice', 25, 'Paris'])
  • items(): Renvoie une vue de toutes les paires clé-valeur. 
    print(personne.items())  # Renvoie toutes les paires clé-valeur

    Sortie :

    dict_items([('nom', 'Alice'), ('age', 25), ('ville', 'Paris')])
  • update(): Met à jour le dictionnaire avec les paires clé-valeur fournies. 
    personne.update({"age": 26, "profession": "ingénieur"})  # Met à jour les valeurs du dictionnaire

    Sortie :

    {'nom': 'Alice', 'age': 26, 'ville': 'Paris', 'profession': 'ingénieur'}
  • pop(): Supprime l'élément avec la clé spécifiée et renvoie sa valeur. 
    age = personne.pop("age")  # Supprime et renvoie la valeur de la clé "age"

    Sortie :

    26
    {'nom': 'Alice', 'ville': 'Paris', 'profession': 'ingénieur'}
  • popitem(): Supprime et renvoie une paire clé-valeur aléatoire du dictionnaire. 
    element_aleatoire = personne.popitem()  # Supprime et renvoie une paire clé-valeur aléatoire

    Sortie :

    ('profession', 'ingénieur')
    {'nom': 'Alice', 'ville': 'Paris'}
  • clear(): Supprime tous les éléments du dictionnaire. 
    personne.clear()  # Supprime tous les éléments

    Sortie :

    {}
  • copy(): Renvoie une copie superficielle du dictionnaire. 
    personne_copie = personne.copy()  # Renvoie une copie du dictionnaire

    Sortie :

    {'nom': 'Alice', 'age': 25, 'ville': 'Paris'}
  • del: Supprime une paire clé-valeur spécifiée. 
    del personne["age"]  # Supprime la paire clé-valeur avec la clé "age"

    Sortie :

    {'nom': 'Alice', 'ville': 'Paris', 'profession': 'ingénieur'}

Ajouter ou remplacer une clé/valeur dans un dictionnaire

personne["profession"] = "ingénieur"  # Ajoute une nouvelle clé/valeur
                    personne["age"] = 26  # Remplace la valeur de la clé "age"

Sortie :

{'nom': 'Alice', 'age': 26, 'ville': 'Paris', 'profession': 'ingénieur'}

Multiplier les valeurs d'un dictionnaire par 2

Il est parfois nécessaire de modifier les valeurs d'un dictionnaire en appliquant une opération à chaque valeur. Pour multiplier toutes les valeurs par 2, vous pouvez utiliser la fonction suivante :


def multiplier_valeurs_par_deux(dictionnaire):
    for cle in dictionnaire:
        dictionnaire[cle] *= 2

Exemple d'utilisation :


a_dictionary = {'John': 12, 'Alex': 8, 'Bob': 14, 'Mike': 14, 'Molly': 16}
multiplier_valeurs_par_deux(a_dictionary)
print(a_dictionary)

Sortie :

{'John': 24, 'Alex': 16, 'Bob': 28, 'Mike': 28, 'Molly': 32}

La fonction multiplier_valeurs_par_deux parcourt chaque clé du dictionnaire et multiplie la valeur correspondante par 2. Ainsi, le dictionnaire initial est modifié avec les nouvelles valeurs.

Autre méthode : création d'un nouveau dictionnaire

Si vous souhaitez conserver le dictionnaire original intact, vous pouvez créer un nouveau dictionnaire avec les valeurs multipliées par 2 :


def multiplier_valeurs_par_deux_nouveau(dictionnaire):
    nouveau_dictionnaire = {cle: valeur * 2 for cle, valeur in dictionnaire.items()}
    return nouveau_dictionnaire

Exemple d'utilisation :


a_dictionary = {'John': 12, 'Alex': 8, 'Bob': 14, 'Mike': 14, 'Molly': 16}
nouveau_dico = multiplier_valeurs_par_deux_nouveau(a_dictionary)
print(nouveau_dico)
print(a_dictionary)  # Le dictionnaire original reste inchangé

Sortie :

{'John': 24, 'Alex': 16, 'Bob': 28, 'Mike': 28, 'Molly': 32}
{'John': 12, 'Alex': 8, 'Bob': 14, 'Mike': 14, 'Molly': 16}

Cette méthode utilise une compréhension de dictionnaire pour créer un nouveau dictionnaire avec les valeurs multipliées par 2, sans modifier le dictionnaire original.

Trouver la clé avec la plus grande valeur entière

Il est souvent utile de trouver la clé associée à la plus grande valeur dans un dictionnaire, en particulier lorsque les valeurs sont des entiers ou des nombres.

Méthode avec la fonction max()

Vous pouvez utiliser la fonction intégrée max() avec le paramètre key pour obtenir la clé ayant la plus grande valeur : 


                        a_dictionary = {'John': 12, 'Alex': 8, 'Bob': 14, 'Mike': 14, 'Molly': 16}
                        cle_max = max(a_dictionary, key=a_dictionary.get)
                        print("La clé avec la plus grande valeur est :", cle_max)

Sortie :

La clé avec la plus grande valeur est : Molly

La fonction max() parcourt les clés du dictionnaire et utilise la méthode get pour comparer les valeurs associées à chaque clé.

Méthode en trouvant la valeur maximale puis la clé associée

Vous pouvez également trouver la valeur maximale puis parcourir le dictionnaire pour trouver la clé associée :


                        a_dictionary = {'John': 12, 'Alex': 8, 'Bob': 14, 'Mike': 14, 'Molly': 16}
                        valeur_max = max(a_dictionary.values())
                        for cle, valeur in a_dictionary.items():
                            if valeur == valeur_max:
                                print("La clé avec la plus grande valeur est :", cle)
                                break

Sortie :

La clé avec la plus grande valeur est : Molly

Cette méthode est utile si plusieurs clés peuvent avoir la même valeur maximale et que vous souhaitez contrôler la manière dont vous gérez ces cas.

Gérer plusieurs clés avec la même valeur maximale

Si plusieurs clés ont la même valeur maximale, vous pouvez les récupérer toutes :


                        a_dictionary = {'John': 16, 'Alex': 8, 'Bob': 14, 'Mike': 16, 'Molly': 16}
                        valeur_max = max(a_dictionary.values())
                        cles_max = [cle for cle, valeur in a_dictionary.items() if valeur == valeur_max]
                        print("Les clés avec la valeur maximale sont :", cles_max)

Sortie :

Les clés avec la valeur maximale sont : ['John', 'Mike', 'Molly']

Cette approche utilise une compréhension de liste pour collecter toutes les clés dont la valeur correspond à la valeur maximale.

Fonction utilitaire

Vous pouvez créer une fonction réutilisable pour cette opération :


                        def trouver_cle_max(dictionnaire):
                            valeur_max = max(dictionnaire.values())
                            cles_max = [cle for cle, valeur in dictionnaire.items() if valeur == valeur_max]
                            return cles_max
                        
                        # Exemple d'utilisation
                        a_dictionary = {'John': 16, 'Alex': 8, 'Bob': 14, 'Mike': 16, 'Molly': 16}
                        cles_max = trouver_cle_max(a_dictionary)
                        print("Les clés avec la valeur maximale sont :", cles_max)

Sortie :

Les clés avec la valeur maximale sont : ['John', 'Mike', 'Molly']

Cette fonction renvoie une liste de clés associées à la valeur maximale, ce qui vous permet de gérer facilement les cas où plusieurs éléments partagent la même valeur élevée.

Exemple avec tri des clés

Si vous souhaitez obtenir la première clé selon un ordre spécifique lorsque plusieurs clés ont la même valeur maximale, vous pouvez trier les clés :


                        a_dictionary = {'John': 16, 'Alex': 8, 'Bob': 14, 'Mike': 16, 'Molly': 16}
                        cle_max = max(sorted(a_dictionary), key=a_dictionary.get)
                        print("La première clé avec la plus grande valeur est :", cle_max)

Sortie :

La première clé avec la plus grande valeur est : John

Ici, les clés sont triées avant d'appliquer la fonction max(), ce qui influence le résultat lorsqu'il y a des ex æquo.

Attention aux types de valeurs

Assurez-vous que les valeurs du dictionnaire sont des nombres entiers (ou du même type) pour éviter les erreurs lors de la comparaison :


                        a_dictionary = {'John': 12, 'Alex': '8', 'Bob': 14}
                        # Ceci provoquera une erreur car '8' est une chaîne de caractères

                        a_dictionary['Alex'] = int(a_dictionary['Alex'])

Multiplier les valeurs d'un dictionnaire par un nombre sans boucle

Il est possible de multiplier les valeurs d'un dictionnaire par un nombre sans utiliser de boucle en utilisant une compréhension de liste.

Fonction utilisant la compréhension de liste

Voici une fonction qui prend un dictionnaire et un nombre, et renvoie une liste contenant toutes les valeurs multipliées par ce nombre :


def multiplier_valeurs_sans_boucle(dictionnaire, nombre):

Multiplier les valeurs d'une liste par un nombre sans boucle

Il est possible de multiplier les valeurs d'une liste par un nombre sans utiliser de boucle, en utilisant la fonction map.

Fonction utilisant map

Voici une fonction qui prend une liste et un nombre, et renvoie une liste contenant toutes les valeurs multipliées par ce nombre :


def multiply_list_map(my_list=[], number=0):
    return list(map(lambda x: x * number, my_list))

Exemple d'utilisation :


my_list = [1, 2, 3, 4, 5]
resultat = multiply_list_map(my_list, 2)
print(resultat)

Sortie :

[2, 4, 6, 8, 10]

La fonction multiply_list_map utilise la fonction map avec une fonction lambda pour multiplier chaque élément de my_list par number, et renvoie le résultat sous forme de nouvelle liste. La liste initiale my_list reste inchangée.

Convertir un chiffre romain en nombre entier

Cette méthode prend en entrée une chaîne de caractères représentant un chiffre romain et renvoie un nombre entier correspondant.

Fonction pour convertir un chiffre romain en nombre entier


                    def roman_to_int(s):
                        romain = {'I': 1, 'V': 5, 'X': 10, 'L': 50, 'C': 100, 'D': 500, 'M': 1000}
                        num = 0
                        for i in range(len(s)):
                            if i > 0 and romain[s[i]] > romain[s[i - 1]]:
                                num += romain[s[i]] - 2 * romain[s[i - 1]]
                            else:
                                num += romain[s[i]]
                        return num

Exemple d'utilisation

                    print(roman_to_int('IV'))      # 4
                    print(roman_to_int('IX'))      # 9
                    print(roman_to_int('LVIII'))   # 58
                    print(roman_to_int('MCMXCIV')) # 1994

                    9
                    58

7. Exemples Avancés

Les collections peuvent être utilisées dans une variété de scénarios plus complexes. Nous allons maintenant examiner quelques exemples avancés de manipulations de collections dans Python.

7.1 Tri et Filtrage d'une Liste d'Objets

Supposons que vous ayez une liste d'objets complexes et que vous souhaitiez trier cette liste ou appliquer un filtre. Vous pouvez le faire facilement avec des fonctions comme sorted() et filter(). 

class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age

    def __repr__(self):
        return f"{self.nom} ({self.age})"

personnes = [
    Personne("Alice", 25),
    Personne("Bob", 30),
    Personne("Charlie", 35)
]

# Trier les personnes par âge
personnes_triees = sorted(personnes, key=lambda p: p.age)
print(personnes_triees)

# Filtrer les personnes de plus de 30 ans
personnes_filtrees = filter(lambda p: p.age > 30, personnes)
print(list(personnes_filtrees))

Sortie :

[Alice (25), Bob (30), Charlie (35)]
[Charlie (35)]

8.3 Utiliser les Générateurs pour Économiser de la Mémoire

Les générateurs sont un excellent moyen d'économiser de la mémoire, surtout lorsque vous travaillez avec des ensembles de données volumineux. Au lieu de charger toute la collection en mémoire, vous pouvez générer les éléments à la demande.

def generate_pairs():
    for x in range(10):
        for y in range(10):
            yield (x, y)

pairs = generate_pairs()
for pair in pairs:
    print(pair)

Les générateurs ne créent pas toute la collection en mémoire à la fois, ce qui les rend très efficaces lorsque vous travaillez avec des données volumineuses.

10. Les Ensembles

Les ensembles (set) sont des collections non ordonnées et non indexées d'éléments uniques. Ils sont utiles pour garantir l'unicité et effectuer des opérations mathématiques comme l'union, l'intersection, ou la différence.

10.1 Création d'un Ensemble

Vous pouvez créer un ensemble en utilisant des accolades {} ou la fonction set() : 

# Avec des accolades
                    ensemble = {1, 2, 3, 4}
                    print(ensemble)
                    
                    # Avec la fonction set()
                    ensemble_vide = set()
                    ensemble_chaine = set("Python")
                    print(ensemble_vide)
                    print(ensemble_chaine)

Sortie :

{1, 2, 3, 4}
                    set()
                    {'n', 'y', 't', 'o', 'P', 'h'}

10.2 Opérations sur les Ensembles

Les ensembles permettent des opérations mathématiques comme :

  • Union (|): Combine tous les éléments de deux ensembles.
  • Intersection (&): Garde les éléments communs à deux ensembles.
  • Différence (-): Soustrait les éléments d'un ensemble à un autre.
  • Différence symétrique (^): Garde les éléments présents dans un seul des deux ensembles.
a = {1, 2, 3}
                    b = {3, 4, 5}
                    
                    print(a | b)  # Union
                    print(a & b)  # Intersection
                    print(a - b)  # Différence
                    print(a ^ b)  # Différence symétrique

Sortie :

{1, 2, 3, 4, 5}
                    {3}
                    {1, 2}
                    {1, 2, 4, 5}

10.3 Méthodes Utiles pour les Ensembles

Voici quelques méthodes utiles pour manipuler les ensembles :

  • add(x): Ajoute un élément à l'ensemble.
  • remove(x): Supprime un élément (erreur si absent).
  • discard(x): Supprime un élément (aucune erreur si absent).
  • pop(): Supprime et retourne un élément arbitraire.
  • clear(): Vide l'ensemble.
ensemble = {1, 2, 3}
                    ensemble.add(4)
                    print(ensemble)
                    
                    ensemble.remove(2)
                    print(ensemble)
                    
                    ensemble.discard(5)  # Pas d'erreur si absent
                    ensemble.pop()  # Élément arbitraire supprimé
                    print(ensemble)
                    
                    ensemble.clear()
                    print(ensemble)

Sortie :

{1, 2, 3, 4}
                    {1, 3, 4}
                    {3, 4}
                    set()
